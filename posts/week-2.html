Week 2 - Progress Update

What We Achieved This Week

This is the second week of our Satellite Orbit and Event Predictor (SOEP) project, and we continued making substantial progress. Here’s a summary of what we accomplished:

  • Server Setup: After reviewing our options, we decided to host our own server instead of using a cloud solution. This will give us more control over the environment and reduce ongoing cloud costs.
  • Docker Implementation: We created a Dockerfile for the frontend application, allowing us to easily build and deploy the frontend in a consistent and isolated environment.
  • Database Setup: We integrated PostgreSQL into our system, setting up the database to store and manage satellite-related data efficiently.
  • Frontend Redesign: We implemented a new frontend design to improve the user experience, focusing on better visual presentation and navigation for our satellite information display.
  • Sgp4Prop Investigation: We delved deeper into the Sgp4Prop (Simplified General Perturbations) library, a key component for our satellite orbit prediction functionality, ensuring a better understanding of how we can use it for more accurate calculations.
